# Polymarket 交易策略 - 实战指南

## 核心原则

**只在一个条件下下注：模型预测概率 > 市场定价概率（存在正边缘）**

---

## 一、快速决策表

### 什么时候下注？

| 剩余时间 | 边缘要求 | 模型概率要求 | 行动 |
|---------|---------|------------|------|
| > 3分钟（早期） | ≥ 5% | ≥ 55% | ✅ 可以下注 |
| 2-3分钟（中期） | ≥ 10% | ≥ 60% | ✅ 可以下注 |
| < 2分钟（后期） | ≥ 20% | ≥ 65% | ⚠️ 谨慎下注 |

### 什么时候不下注？

- ❌ 边缘 < 阈值
- ❌ 模型概率 < 最小概率
- ❌ 技术指标互相矛盾
- ❌ 市场状态 = CHOP（横盘）
- ❌ 流动性 < 1,000

---

## 二、具体下注信号

### 信号1：做多（买入"上涨"）

**条件（必须同时满足）：**

1. **边缘（上涨）≥ 阈值**
   - 早期（>3分钟）：≥ 5%
   - 中期（2-3分钟）：≥ 10%
   - 后期（<2分钟）：≥ 20%

2. **模型预测上涨概率 ≥ 最小概率**
   - 早期（>3分钟）：≥ 55%
   - 中期（2-3分钟）：≥ 60%
   - 后期（<2分钟）：≥ 65%

3. **技术指标至少3个支持看涨：**
   - ✅ 价格 > VWAP
   - ✅ VWAP斜率 > 0（上升）
   - ✅ RSI > 55 且 RSI斜率 > 0
   - ✅ MACD柱状图 > 0 且扩大
   - ✅ Heiken Ashi 绿色且连续 ≥ 2根
   - ✅ 1分钟和3分钟价格变化都是正值

4. **市场状态 ≠ CHOP（横盘）**

5. **价格差距合理：**
   - 当前价格 < 目标价格，但差距 < 1%
   - 或者当前价格 > 目标价格

**示例场景：**
```
剩余时间：4分钟（早期）
模型预测：上涨 65%，下跌 35%
市场定价：上涨 50¢（50%），下跌 50¢（50%）
边缘（上涨）= 65% - 50% = 15% ✅
技术指标：价格>VWAP, VWAP上升, RSI 58↑, MACD看涨扩大 ✅
市场状态：TREND_UP ✅
→ 下注：买入"上涨"
```

---

### 信号2：做空（买入"下跌"）

**条件（必须同时满足）：**

1. **边缘（下跌）≥ 阈值**
   - 早期（>3分钟）：≥ 5%
   - 中期（2-3分钟）：≥ 10%
   - 后期（<2分钟）：≥ 20%

2. **模型预测下跌概率 ≥ 最小概率**
   - 早期（>3分钟）：≥ 55%
   - 中期（2-3分钟）：≥ 60%
   - 后期（<2分钟）：≥ 65%

3. **技术指标至少3个支持看跌：**
   - ✅ 价格 < VWAP
   - ✅ VWAP斜率 < 0（下降）
   - ✅ RSI < 45 且 RSI斜率 < 0
   - ✅ MACD柱状图 < 0 且扩大
   - ✅ Heiken Ashi 红色且连续 ≥ 2根
   - ✅ 1分钟和3分钟价格变化都是负值
   - ✅ VWAP收复失败（强烈信号，+3分）

4. **市场状态 ≠ CHOP（横盘）**

5. **价格差距合理：**
   - 当前价格 > 目标价格，但差距 < 1%
   - 或者当前价格 < 目标价格

**示例场景：**
```
剩余时间：2.5分钟（中期）
模型预测：上涨 35%，下跌 65%
市场定价：上涨 40¢（40%），下跌 60¢（60%）
边缘（下跌）= 65% - 60% = 5% ❌（需要≥10%）
→ 不下注（边缘不足）

但如果：
市场定价：上涨 30¢（30%），下跌 70¢（70%）
边缘（下跌）= 65% - 70% = -5% ❌（负边缘）
→ 不下注（市场已经高估下跌概率）
```

## 三、不同时间段的策略

### 早期阶段（剩余时间 > 3分钟）

**策略：积极寻找机会**

**下注条件：**
- 边缘 ≥ 5%
- 模型概率 ≥ 55%
- 技术指标至少2个支持

**优势：**
- 有足够时间让价格回归
- 可以承受较小的边缘
- 容错率较高

**风险：**
- 价格可能反转
- 需要快速决策

**示例：**
```
剩余时间：4分钟
模型：上涨 60%，下跌 40%
市场：上涨 50¢，下跌 50¢
边缘：10%
技术指标：价格>VWAP, VWAP上升, RSI 56↑
→ ✅ 下注做多
```

---

### 中期阶段（2-3分钟）

**策略：精选机会**

**下注条件：**
- 边缘 ≥ 10%
- 模型概率 ≥ 60%
- 技术指标至少3个支持
- 市场状态必须是趋势（TREND_UP或TREND_DOWN）

**优势：**
- 趋势更明确
- 时间压力适中

**风险：**
- 时间开始紧迫
- 需要更强的信号

**示例：**
```
剩余时间：2.5分钟
模型：上涨 70%，下跌 30%
市场：上涨 55¢，下跌 45¢
边缘：15%
技术指标：价格>VWAP, VWAP上升, RSI 62↑, MACD看涨扩大, HA绿色x5
市场状态：TREND_UP
→ ✅ 下注做多
```

---

### 后期阶段（< 2分钟）

**策略：极度谨慎，只做高确定性交易**

**下注条件：**
- 边缘 ≥ 20%（非常高的边缘）
- 模型概率 ≥ 65%
- 技术指标至少4个强烈支持
- 价格差距 < 0.3%（接近目标价格）
- 市场状态必须是强趋势

**优势：**
- 结果很快揭晓
- 如果信号强烈，胜率可能很高

**风险：**
- 时间极短，价格变化空间有限
- 如果价格差距大，几乎不可能反转
- 滑点可能较大

**示例：**
```
剩余时间：1.5分钟
模型：上涨 75%，下跌 25%
市场：上涨 50¢，下跌 50¢
边缘：25%
技术指标：价格>VWAP, VWAP上升, RSI 68↑, MACD看涨扩大, HA绿色x8, 价格变化+$50
价格差距：当前$69,590 vs 目标$69,598（差距$8，0.01%）
市场状态：TREND_UP
→ ✅ 可以下注做多（但仓位要小）

但如果：
价格差距：当前$69,479 vs 目标$69,598（差距$119，0.17%）
→ ❌ 不下注（差距太大，1.5分钟不可能涨$119）
```

## 四、特殊情况处理

### 情况1：技术指标矛盾

**场景：**
- RSI < 30（超卖，可能反弹）
- 但MACD看跌且扩大
- 价格 < VWAP

**处理：**
- 如果边缘足够大（≥ 15%），可以下注
- 如果边缘小（< 10%），不下注
- 等待指标一致

---

### 情况2：VWAP收复失败

**场景：**
- 价格尝试回到VWAP但失败
- 这是强烈的看跌信号（+3分）

**处理：**
- 如果边缘（下跌）≥ 阈值 → 立即做空
- 这是最强烈的看跌信号之一

---

### 情况3：市场已经充分定价

**场景：**
- 模型预测：上涨 60%
- 市场定价：上涨 58¢（58%）
- 边缘只有 2%

**处理：**
- ❌ 不下注（边缘太小）
- 等待下一个5分钟窗口
- 或者寻找其他市场

---

### 情况4：超卖/超买反弹

**场景：**
- RSI < 30（超卖）
- 但其他指标都看跌
- 边缘（下跌）很大

**处理：**
- **谨慎**：RSI超卖可能反弹
- 如果剩余时间 > 2分钟，可以等待RSI斜率转正
- 如果剩余时间 < 2分钟，跟随主要趋势（其他指标）

---

### 情况5：价格差距极大

**场景：**
- 当前价格：$69,479
- 目标价格：$69,598
- 差距：$119（0.17%）
- 剩余时间：2分钟

**处理：**
- ❌ **绝对不下注做多**（不可能在2分钟内涨$119）
- ✅ 可以考虑做空（如果其他条件满足）
- 或者等待下一个窗口

**经验法则：**
- 剩余时间 < 1分钟，价格差距 > 0.05% → 几乎不可能反转
- 剩余时间 < 2分钟，价格差距 > 0.1% → 非常困难
- 剩余时间 > 3分钟，价格差距 < 0.15% → 可能

## 五、实际交易流程

### 步骤1：检查基础条件

```
1. 剩余时间是多少？
2. 流动性是否足够（> 1,000）？
3. 市场状态是什么（TREND/RANGE/CHOP）？
4. 如果CHOP → 不下注
```

### 步骤2：计算边缘

```
1. 模型预测概率是多少？
2. 市场定价概率是多少？
3. 边缘 = 模型预测 - 市场定价
4. 边缘是否 ≥ 阈值？
   - 早期：≥ 5%
   - 中期：≥ 10%
   - 后期：≥ 20%
```

### 步骤3：检查技术指标

```
统计支持看涨/看跌的指标数量：

看涨指标：
- 价格 > VWAP (+1)
- VWAP斜率 > 0 (+1)
- RSI > 55 且斜率 > 0 (+1)
- MACD看涨且扩大 (+1)
- Heiken Ashi 绿色且连续 ≥ 2 (+1)
- 价格变化都是正值 (+1)

看跌指标：
- 价格 < VWAP (+1)
- VWAP斜率 < 0 (+1)
- RSI < 45 且斜率 < 0 (+1)
- MACD看跌且扩大 (+1)
- Heiken Ashi 红色且连续 ≥ 2 (+1)
- 价格变化都是负值 (+1)
- VWAP收复失败 (+3，强烈信号)

要求：至少3个指标支持同一方向
```

### 步骤4：检查价格差距

```
1. 当前价格 vs 目标价格
2. 差距是否合理？
   - 剩余时间 > 3分钟：差距 < 0.15% 可以
   - 剩余时间 2-3分钟：差距 < 0.1% 可以
   - 剩余时间 < 2分钟：差距 < 0.05% 可以
```

### 步骤5：最终决策

```
如果所有条件满足：
- ✅ 下注

如果任何条件不满足：
- ❌ 不下注，等待下一个窗口
```

## 七、仓位管理

### 仓位大小建议

| 边缘大小 | 信号强度 | 剩余时间 | 建议仓位 |
|---------|---------|---------|---------|
| ≥ 20% | STRONG | 任何 | 标准仓位 |
| 10-20% | GOOD | > 2分钟 | 标准仓位 |
| 10-20% | GOOD | < 2分钟 | 减半仓位 |
| 5-10% | OPTIONAL | > 3分钟 | 减半仓位 |
| 5-10% | OPTIONAL | < 3分钟 | 不下注 |

### 风险控制

1. **单次下注不超过总资金的10%**
2. **如果连续3次亏损，暂停交易**
3. **如果剩余时间 < 1分钟，最多只用5%仓位**
4. **如果价格差距 > 0.1%，不下注**

## 八、总结：我的下注规则

### ✅ 我会下注当：

1. **边缘 ≥ 阈值**（根据时间调整）
2. **模型概率 ≥ 最小概率**（根据时间调整）
3. **至少3个技术指标支持同一方向**
4. **市场状态 ≠ CHOP**
5. **价格差距合理**（根据剩余时间判断）
6. **流动性充足**（> 1,000）

### ❌ 我不会下注当：

1. **边缘 < 阈值**
2. **模型概率 < 最小概率**
3. **技术指标矛盾**（< 3个支持同一方向）
4. **市场状态 = CHOP**
5. **价格差距太大**（剩余时间 < 1分钟，差距 > 0.05%）
6. **流动性不足**（< 1,000）
7. **任何不确定的情况**

### 🎯 最佳下注时机：

- **早期阶段（> 3分钟）+ 边缘 ≥ 10% + 技术指标强烈支持**
- **中期阶段（2-3分钟）+ 边缘 ≥ 15% + 趋势明确**
- **后期阶段（< 2分钟）+ 边缘 ≥ 20% + 价格差距 < 0.05%**

---

## 九、记住这些关键点

1. **边缘是核心**：没有边缘，不下注
2. **时间很重要**：剩余时间越少，要求越高
3. **一致性**：多个指标指向同一方向，信号更强
4. **价格差距**：差距太大，即使信号再强也不下注
5. **风险管理**：永远不要下注超过你能承受的损失

**最后：如果犹豫，就不下注。等待下一个5分钟窗口。**
